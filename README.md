# Paola García - Portfolio

Personal portfolio of **Paola García**, actress and dubbing director. Single-page application (SPA) built with React + Vite.

**Deploy Pending** - domain pending.

## Tech Stack:

- **React** + **Vite**
- **Pure CSS** - no frameworks, custom BEM methodology
- **yet-another-react-lightbox** - gallery with lightbox modal
- **EmailJS** - contact form without backend
- **React Icons** - iconography
- **ImageKit CDN** - optimized image delivery

## Features

- **SPA with smooth scroll** — anchor-based navigation between sections
- **Fixed Header** — transparent → solid effect on scroll, with custom `useScroll` hook
- **Responsive hamburger menu** — mobile navigation at 768px breakpoint
- **Hero section** — full-screen image with fluid typography via `clamp()`
- **About section** — two-column grid layout with achievements and icons
- **Gallery** — photo grid with lightbox modal (yet-another-react-lightbox)
- **Projects** — filterable cards by category (Actuación / Doblaje)
- **Contact form** — powered by EmailJS, no backend required
- **Fully responsive** — mobile, tablet, and desktop
- **Fluid typography** throughout with CSS `clamp()`
