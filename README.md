# Paola García - Portfolio

Personal portfolio of **Paola García**, actress and dubbing director. Single-page application (SPA) built with React + Vite.

## Tech Stack:

- **React** + **Vite**
- **Pure CSS** - no frameworks
- **yet-another-react-lightbox** - gallery with lightbox
- **EmailJS** - contact form without backend
- **React Icons** - icons

## Features

- Smooth scroll navigation between sections
- Header with transparent -> solid effect on scroll
- Responsive hamburger menu
- Gallery with lightbox modal
- Project filter by category
- Contact form powered by EmailJS
- Fully responsive design (mobile, tablet, desktop)
- Fluid typography with `clamp()`
